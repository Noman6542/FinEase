# FinEase — Full Stack Finance Management App

**FinEase** is a modern full-stack financial management web application that helps users manage their daily transactions effortlessly.  
It provides an intuitive interface for users to **add**, **update**, **view**, and **delete** transactions, powered by a secure and scalable backend.


## Tech Stack

### Frontend
 **React** – Component-based UI library  
 **Tailwind CSS** – Utility-first CSS framework  
 **DaisyUI** – Tailwind-based UI component library  
 **Firebase** – Authentication & hosting  
 **React Toastify** – Toast notifications  
 **React Spinners** – Loading animations  
 **Google Fonts** – Beautiful typography  

### Backend
 **Node.js** – Runtime environment  
 **Express.js** – Backend web framework  
 **CORS** – Handle cross-origin requests  
 **Nodemon** – Auto-restart during development  
 **MongoDB (optional)** – Database for storing data  
 **dotenv** – Manage environment variables
